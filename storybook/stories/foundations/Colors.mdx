## Best Practices

- Use semantic color classes (primary, success, danger) for consistent theming
- Ensure sufficient color contrast for accessibility (WCAG AA compliance)
- Use muted colors for secondary information
- Avoid using color alone to convey meaning - combine with icons or text

## Accessibility Notes

- All color combinations meet WCAG AA contrast requirements
- Color is not the only way information is conveyed
- Consider users with color vision deficiencies
- Test color combinations in different lighting conditions
