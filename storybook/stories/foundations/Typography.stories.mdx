# Typography

This section showcases the typography tokens and utilities available in the design system.

## Typography Tokens

The following typography tokens are available from the Digdir Design System:

- `--digdir-font-size-xs`: Extra small text
- `--digdir-font-size-sm`: Small text
- `--digdir-font-size-md`: Medium text
- `--digdir-font-size-lg`: Large text
- `--digdir-font-size-xl`: Extra large text

- `--digdir-font-weight-normal`: Normal font weight
- `--digdir-font-weight-bold`: Bold font weight

- `--digdir-line-height-tight`: Tight line height
- `--digdir-line-height-normal`: Normal line height
- `--digdir-line-height-relaxed`: Relaxed line height

## Best Practices

- Use semantic heading hierarchy (h1-h6) for proper document structure
- Maintain consistent font sizes and weights across similar content types
- Choose appropriate line heights based on content length and reading context
- Consider responsive typography for different screen sizes
- Use text transformation sparingly and purposefully

## Accessibility Notes

- Ensure sufficient color contrast between text and background
- Maintain adequate font sizes for readability (minimum 14px for body text)
- Use semantic HTML elements for proper screen reader navigation
- Avoid using color alone to convey meaning
- Test typography in high contrast mode
- Consider line length (50-75 characters) for optimal reading
