# Spacing

This section showcases the spacing tokens and utilities available in the design system.

## Spacing Tokens

The following spacing tokens are available from the Digdir Design System:

- `--digdir-spacing-xs`: Extra small spacing
- `--digdir-spacing-sm`: Small spacing
- `--digdir-spacing-md`: Medium spacing
- `--digdir-spacing-lg`: Large spacing
- `--digdir-spacing-xl`: Extra large spacing

### Margin Classes

```css
.m-0 { margin: 0; }
.m-xs { margin: var(--digdir-spacing-xs); }
.m-sm { margin: var(--digdir-spacing-sm); }
.m-md { margin: var(--digdir-spacing-md); }
.m-lg { margin: var(--digdir-spacing-lg); }
.m-xl { margin: var(--digdir-spacing-xl); }

.mt-0 { margin-top: 0; } /* Top margin */
.mr-0 { margin-right: 0; } /* Right margin */
.mb-0 { margin-bottom: 0; } /* Bottom margin */
.ml-0 { margin-left: 0; } /* Left margin */
```

## Best Practices

- Use consistent spacing scale across your application
- Prefer margin/padding utilities over custom CSS for maintainability
- Use larger spacing for important content separation
- Consider responsive spacing needs for different screen sizes

## Accessibility Notes

- Adequate spacing improves readability and usability
- Ensure interactive elements have sufficient touch targets (minimum 44px)
- Spacing helps with visual hierarchy and content organization
- Consider spacing for keyboard navigation and focus indicators
